Mjolnir Archive Service

Application is successfully deployed.

Control Endpoints

/archive-users
Triggers the archiving job - archives repositories and removes memberships of previously discovered offboarded users.
/delete-old-archives
Deletes archived repositories which are older than specified limit and are no longer needed.
/ldap-scan
Discover offboarded users by comparing user database with company LDAP.
/generate-email-report
Generates report that lists performed removals, discovered unknown users, etc.
/generate-test-email
Generates empty test email.
/update-github-usernames
Updates GH usernames of registered usres according to their GH IDs.
